Nivo Slider

What is Nivo Slider

Nivo Slider is a tool to use for changing images on a page. You can utilize it in many ways. Having different call to actions on a page rotate through automatically at a set interval, or you can use it as a gallery of images and have the user advance to the next one. I personally like this tool because it is easy to use and get up and going on any site. All you have to do is add a couple js files, update some classes, and start the function.

The Code

My main.js files looks like:

$(window).load(function() { $('#slider').nivoSlider(); }); This will start the function that is located in the jquery.nivo.slider.js file.

Inside the jquery.nivo.slider.js file there is lot of code, but the part that I chnaged is below. It shows the different options you can use and you can also change these options for each slide instead of gloabally changing them. You would go to the index.html file and add it there.

//Default settings $.fn.nivoSlider.defaults = { effect: 'random', slices: 15, boxCols: 8, boxRows: 4, animSpeed: 500, pauseTime: 3000, startSlide: 0, directionNav: true, controlNav: true, controlNavThumbs: true, pauseOnHover: true, manualAdvance: false, prevText: 'Prev', nextText: 'Next', randomStart: true, beforeChange: function(){}, afterChange: function(){}, slideshowEnd: function(){}, lastSlide: function(){}, afterLoad: function(){} };
